BSTIMAPI BResult testModules ( BTestModules *  output)

Used as a way to diagnose the current modules that reside in the stimulator to determine if there output voltage levels are okay and ensure that they continue to function at the right levels over time. A know stimulus is applied to a known load and the voltage is compared to what it should be.

Parameters
[in,out]outputPointer to a BTestModules strucutre which will contain the status of all the current modules
Returns
a BResult status, either hardware or software error
